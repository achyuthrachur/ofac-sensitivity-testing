**Critical details:**
- `key={stat.label}` moves to `StatTiltCard` (the new outermost element in the map)
- `StatTiltCard className="rounded-xl"` — the perspective container has rounded corners so the perspective clip is soft; no overflow-hidden on this div
- Inner div changes `bg-white/10` → `bg-white/15` (user decision: upgrade opacity so tilt reads well on dark background)
- Inner div adds `[transform-style:preserve-3d]` so content participates in the 3D space
- The `.stat-value`, `.stat-number`, `.stat-prefix`, `.stat-suffix` classNames are UNCHANGED — FeatureStatsAnimationShell's querySelector selectors will continue to find them
- `rotateAmplitude` is not passed (defaults to 7 inside StatTiltCard — the 5–10° professional range)

**Add import at top of file:**
```tsx
import { StatTiltCard } from '@/components/ui/stat-tilt-card';
```

FeatureStatsSection.tsx must remain a Server Component — do NOT add `'use client'`. StatTiltCard itself has `'use client'` and is a Client Component island within this Server Component.
