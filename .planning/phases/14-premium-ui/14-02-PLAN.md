Create a new `'use client'` component at `src/app/_components/landing/ClientHeroHeadline.tsx`. This component:
1. Renders the `<h1>` headline via BlurText with `as="h1"` — this preserves semantic HTML while using BlurText's motion.span children internally
2. Manages `headlineDone` state via `useState(false)`, set to `true` via `onAnimationComplete`
3. Fades in the subtitle `<p>` using `motion.p` from `'motion/react'` — initial opacity 0, animate to 1 when `headlineDone === true`

**Exact text strings** (copy verbatim — these are the production copy):
- Headline: `"Test your OFAC screening before your client does."`
- Subtitle: `"A live sensitivity-testing tool that degrades real-world name variations against 285 synthetic SDN entries. No file prep. No waiting."`

**className values** — match the classes currently on HeroSection.tsx's h1 and p exactly:
- BlurText className: `"text-4xl md:text-5xl lg:text-6xl text-white font-bold max-w-3xl leading-tight mx-auto justify-center"`
- motion.p className: `"text-lg text-white/80 mt-6 max-w-xl mx-auto"`

**Animation params:**
- `animateBy="words"` — splits on spaces (9 words total)
- `delay={90}` — 90ms between words; word 9 starts at 810ms, total ~1160ms
- `direction="top"` — words blur in from slightly above
- `stepDuration={0.35}` — 350ms per blur-to-sharp step

**Subtitle fade:**
- `initial={{ opacity: 0 }}`
- `animate={{ opacity: headlineDone ? 1 : 0 }}`
- `transition={{ duration: 0.6, ease: 'easeOut' }}`

```tsx
'use client';

import { useState } from 'react';
import { motion } from 'motion/react';
import BlurText from '@/components/ui/blur-text';

export function ClientHeroHeadline() {
  const [headlineDone, setHeadlineDone] = useState(false);

  return (
    <>
      <BlurText
        text="Test your OFAC screening before your client does."
        animateBy="words"
        delay={90}
        direction="top"
        stepDuration={0.35}
        as="h1"
        onAnimationComplete={() => setHeadlineDone(true)}
        className="text-4xl md:text-5xl lg:text-6xl text-white font-bold max-w-3xl leading-tight mx-auto justify-center"
      />
      <motion.p
        className="text-lg text-white/80 mt-6 max-w-xl mx-auto"
        initial={{ opacity: 0 }}
        animate={{ opacity: headlineDone ? 1 : 0 }}
        transition={{ duration: 0.6, ease: 'easeOut' }}
      >
        A live sensitivity-testing tool that degrades real-world name variations against 285 synthetic
        SDN entries. No file prep. No waiting.
      </motion.p>
    </>
  );
}
```
